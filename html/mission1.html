-- Mission 1 --

  • Contexte

    Notre première mission consistait à étudier la base de données sur le logiciel “Excel” de l’entreprise Kdou. Pour cela, nous avons répondu à plusieurs questions en usant des propriétés d’Excel ( tri par ordre croissant/décroissant, alphabétique/ de Z à A, ou encore la recherche de mot précis). Comme ceci fut nos premiers travaux sur des bases de données, le but principal de cette mission était surtout de se familiariser avec l’exploitation des bases de données, en distinguant les différents types de clés primaires et secondaires.

  • Bilan

    Excel s’est révélé être difficile lors de l’étude des données : pour la plupart des questions il a fallu lire les tables pour pouvoir y répondre car impossible de formuler des requêtes. De plus, la base de données sur Excel est incorrecte : présence d’aucune clé primaire dans certaines tables (par exemple dans la table “ DetailCommande”), incohérence entre les données et les tables (l’attribut “Prix unitaire” est retrouvable dans deux tables mais les valeurs ne se correspondent pas)

    (voir illustration en dessous)

  • Perspective d’amélioration, conclusion

    Pour conclure, Excel n’est pas le bon logiciel lorsqu’il s’agit d’étude de bases de données car les tables ne sont pas directement liées entre elles ( il est possible de supprimer n’importe quelle données ou de la modifier). De plus, pour des recherches précises cela prend du temps car nous devons lire chaque données ( comme dit précedemment, impossible de formuler des requêtes).
